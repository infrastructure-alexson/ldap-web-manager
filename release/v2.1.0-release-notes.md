## Migration Guide

### From v2.0.0 to v2.1.0

1. **Backup Data**
   ```bash
   pg_dump ldap_manager > backup.sql
   ```

2. **Update Dependencies**
   ```bash
   pip install -r backend/requirements.txt
   npm install
   ```

3. **Run Migrations**
   ```bash
   cd backend
   alembic upgrade head
   ```

4. **Test**
   ```bash
   npm run dev      # Frontend
   uvicorn app.main:app --reload  # Backend
   ```
